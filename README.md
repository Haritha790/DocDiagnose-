🩺 DocDiagnose – AI-Based Disease Prediction & Doctor Recommendation System

DocDiagnose is a smart healthcare web application developed using Flask, MongoDB, and Machine Learning. The system predicts possible diseases based on symptoms entered by users and recommends suitable doctors based on specialization and location.

It provides a simple and user-friendly interface for patients to:

Register/Login securely
Enter symptoms
Get disease predictions
Find nearby doctors
Book appointments
🚀 Features
🔐 User Authentication (Login & Registration)
🤖 Disease Prediction using Machine Learning
🩺 Doctor Recommendation System
📍 Location-based Doctor Search
📅 Appointment Booking
💾 MongoDB Database Integration
🌐 Responsive Web Interface using Flask

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Haritha790/DocDiagnose-.git
cd DocDiagnose-
2️⃣ Create Virtual Environment
python -m venv venv
3️⃣ Activate Virtual Environment
Windows
venv\Scripts\activate
Linux/Mac
source venv/bin/activate
4️⃣ Install Dependencies
pip install -r requirements.txt
5️⃣ Run the Flask Application
python app.py
🌐 Output

After running the project, open:

http://127.0.0.1:5000
